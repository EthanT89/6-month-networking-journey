# Phase 3 — Lighting

**Goal:** Implement the Phong lighting model. Make surfaces look like materials.

## Objectives

- [ ] Surface normals — per-vertex, correct transform (transpose of inverse)
- [ ] Ambient term
- [ ] Diffuse lighting (Lambertian reflectance)
- [ ] Specular highlights — Phong and Blinn-Phong
- [ ] Point light with attenuation
- [ ] Directional light
- [ ] Multiple light sources

## Checkpoint

A sphere lit from one side with a visible specular highlight that moves when the camera rotates.
